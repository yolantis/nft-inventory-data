# nft-inventory-data
script to keep track of available nfts, and it has a filter for blockchain and to mark items as sold.
For every entry, upload the social media version of the NFT to the yolantisart media library and add the "https" to the URL from the autocopy button.
When you add more entries, these date formats will work:

✅ "November 2024"
✅ "2024-11-15"
✅ "Nov 15, 2024"

status attributes are case-sensitive: 1:1 available, edition partially sold, edition available, sold
blockchain attributes: Base ETH, Tezos, ETH

! problem with the preview of videos minted on tezos. You'll have to add a screenshot to the media library instead of letting it play in WP.
